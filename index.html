Overview

What is SecureVision 90?

Most surveillance systems are either passive CCTV cameras that only record incidents, or expensive AI solutions requiring cloud servers and constant internet connectivity.

SecureVision 90 solves this with an affordable, offline-capable, AI-driven surveillance system running entirely on a Raspberry Pi 5. It handles real-time person detection, face recognition, weapon identification, and fight detection — all locally.

🔒

Privacy First

All processing happens on-device — no data leaves the premises.

⚡

Real-Time Alerts

Instant dashboard notifications with timestamped alert logs.

💰

Cost Effective

Fraction of commercial AI camera system costs.

Capabilities

Core Features

Four powerful AI modules working in real-time harmony

Person Detection

Real-time multi-person detection with bounding boxes and confidence scores. Built on YOLOv8 for blazing-fast inference.

~90% Accuracy

Face Recognition

Identifies known vs. unknown individuals using deep face embeddings. Registers faces and triggers alerts for strangers.

Known / Unknown

Weapon Detection

Detects dangerous objects like scissors, knives, and other weapons in real-time. Triggers immediate color-coded alerts.

⚠ Threat Detected

Fight / Aggression Detection

Skeleton-based pose analysis detects aggressive movements and fights between individuals. Powered by MediaPipe.

System Design

How It Works

📷
Camera Input
Logitech webcam streams live video to Raspberry Pi 5
→
🧠
AI Processing
YOLO, MediaPipe & DeepFace run in parallel threads
→
🔍
Detection
Person, face, weapon & fight events identified
→
📡
Socket.IO
Events streamed in real-time to web dashboard
→
📊
Dashboard
Live feed + alert log on Raspberry Pi's TFT screen

🎯 Accuracy

  • Person Detection: ~90%
  • Face Recognition: ~85%
  • Weapon Detection: ~88%
  • Fight Detection: ~80%

⚡ Performance

  • Processing: 3–7 FPS
  • Latency: < 300ms
  • Platform: Raspberry Pi 5
  • OS: Raspberry Pi OS

🔧 Hardware

  • MCU: Raspberry Pi 5
  • Camera: Logitech C270
  • Display: 3.5" TFT LCD
  • Storage: MicroSD 64GB

🌐 Connectivity

  • Mode: Fully Offline
  • Protocol: WebSocket (LAN)
  • Dashboard: Browser-based
  • Cloud: Not Required
